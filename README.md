# Desafio 3 - Compass UOL

## Descrição do Desafio

-   O desafio consiste em recriar um site de compras de móveis, mantendo-se fiel ao seu design original utilizando HTML, CSS (se necessário), Javascript + Typescript, React e um framework da sua escolha (Bootstrap, Tailwind).
-   Na apresentação, o projeto precisa estar rodando em uma instância EC2.
-   Todas as imagens devem estar hospedadas em um Bucket do S3.

## Tecnologias Utilizadas

As tecnologias utilizadas neste projeto são:

-   **HTML**: Linguagem de marcação para construir a estrutura da página web.
-   **TypeScript**: Superset de JavaScript que adiciona tipagem estática à linguagem.
-   **React**: Biblioteca JavaScript para construir interfaces de usuário.
-   **React DOM**: Pacote que fornece métodos específicos do DOM para aplicações React.
-   **React Router DOM**: Biblioteca para gerenciar rotas em aplicações React.
-   **ESLint**: Ferramenta de linting para identificar e corrigir problemas em código JavaScript.
-   **Vite**: Ferramenta de build rápida e leve para projetos web modernos.
-   **Tailwind CSS**: Framework CSS utilitário para criar layouts personalizados de forma rápida.
-   **AWS EC2**: Serviço de computação em nuvem da Amazon.
-   **AWS S3**: Serviço de armazenamento em nuvem da Amazon.

## Como Rodar o Projeto

### 1. Fazer um fork do projeto

### 2. Clonar o projeto

```bash
git clone https://github.com/Tayling-Ng/desafio-4_compassUol.git
```

### 3. Instalar as dependências

```bash
npm install
```

### Rodar a aplicação

```bash
npm run dev
```
